# Prompt 玄学检测器

---

## 这是什么？

你见过的 Prompt 技巧，有些确实有用，有些只是听起来厉害。

玄学检测器帮你分辨：你写的 Prompt 里，哪些部分**对 AI 输出质量没有稳定帮助**，以及应该换成什么。

---

## 怎么玩？

把你想检测的 Prompt 粘贴过来，系统会逐段标注：

```
命中片段：「你是一个顶级专家，请深吸一口气，充分思考」
├─ 玄学类型：空泛角色 + 仪式型
├─ 为什么不稳定：
│   这些词不提供任何"任务对象 / 输入 / 输出 / 验收"的约束。
│   模型不会因为这些话"更努力"，只会因为"拿到更清楚的指令"才变好。
└─ 可以替换成：
    把「顶级专家」→ 换成「受众是 CTO，重点呈现 ROI 与可行性」
    把「深吸一口气 / 充分思考」→ 删掉，换成「请按 现状 → 冲突 → 结论 三步输出」
```

---

## 会检测这些

| 玄学类型 | 常见说法 |
|----------|----------|
| 仪式型 | 深吸一口气、请仔细想想、Let's think step by step |
| 空泛角色 | 你是顶级专家、你是资深大师 |
| 重要性强调 | 这很重要、请务必、关键任务 |
| 利诱型 | 给你小费、请你喝咖啡、谢谢 |
| 堆砌质量词 | 全面、深入、高质量、专业 |
| 假逻辑框架 | 用第一性原理分析、用苏格拉底法提问 |

---

## 核心认知

> 这些词读起来很专业，但它们没有告诉 AI 任何关于你任务的事实。
> 真正的 Prompt 不是咒语，是规格书。

检测完玄学之后，你会进入**改写训练**——
学习怎么把"玄学空间"变成"任务槽位"。
