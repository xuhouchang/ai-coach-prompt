## 6 类任务结构图卡

### 1. 对比型：7 个节点

`决策问题 → 对比对象 → 输入材料 → 对比维度 → 使用场景/读者 → 输出格式 → 验收标准`

例：不是“分析竞品”，而是先回答“这次比较要帮助谁做什么决策”，再限定比较哪几家、依据哪些资料和维度。

### 2. 生成型：6 个节点

`要交付什么 → 已知背景/素材 → 给谁看 → 内容要点与边界 → 格式/语气 → 验收标准`

例：写客户邮件时，先说明邮件目标、会议背景、收件人、必须覆盖的内容、语气和能否直接发送。

### 3. 抽取型：5 个节点

`源材料 → 要抽什么字段 → 过滤/准确边界 → 输出结构 → 负责人/截止信息`

### 4. 评估型：6 个节点

`评估对象 → 评估标准 → 目标客户/场景 → 风险边界 → 打分/排序规则 → 建议格式`

### 5. 改写型：6 个节点

`原文 → 目标读者 → 改写目的 → 必须保留 → 必须删除/改变 → 输出格式`

### 6. 多步骤型：6 个节点

`目标范围 → 信息来源 → 步骤顺序 → 每步中间产物 → 判断标准 → 最终决策格式`

> 记住：节点数量不是死规则。它的作用是让你发现 AI 还需要猜什么；只要关键节点未定义，先补节点，再要求输出。
