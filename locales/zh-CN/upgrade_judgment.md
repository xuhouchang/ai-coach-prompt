# 任务升级判断

---

## 这是什么？

有些任务，不是 Prompt 写得不好，而是**它超出了单条 Prompt 的能力范围**。

这个判断帮你看清：你的任务到底该用什么级别的工具。

---

## 四个信号

以下任一信号出现，就说明这个任务不适合用单条 Prompt 解决：

| 信号 | 比如 | 应该用什么 |
|------|------|-----------|
| 多步骤 | "先…再…然后…" | Workflow |
| 外部数据 | "查一下最新的 / 调用 API" | MCP |
| 状态保存 | "记住 / 下次接着 / 跟踪" | Skill / Agent |
| 多角色 | "让 A 审 B / 审批流程" | Agent |

---

## 能力演进链

```
📝 Prompt   → 单条任务指令：把一件事说清楚
     ↓
🔗 Workflow → 多步节点串联：把一串事按顺序跑
     ↓
🧩 Skill    → 可复用能力封装：带参数、可反复调用
     ↓
🤖 Agent    → 自主规划 + 调用工具 + 多角色协作
```

---

## 接下来学什么？

根据命中的信号，系统会推荐对应的课程：

- **外部数据依赖** → 先学 MCP（让 AI 用工具）
- **多步骤任务** → 再学 Workflow（节点拆解）
- **状态保存 / 多角色** → 学 Agent（自主规划 + 协作）

---

## 核心认知

> Prompt 是地基，不是天花板。
> 当一条 Prompt 装不下任务时，不是你写错了——
> 是任务需要更大的工具。
> 这就是下一阶段学习的内容。
