# Trae adapter

Copy this complete directory to `.trae/skills/ai-coach-prompt/` in a project, or to Trae's configured global skills directory. Keep `SKILL.md` at the skill root.

Trae may ignore files it does not use. Before practice, use its current session
capability in this order: safe HTML/widget → Mermaid → `course_map.md` Markdown
diagram. Use Trae's native selection controls when exposed; otherwise use the
labelled-letter fallback. The core uses relative paths, so no Trae-only copy is
required.
