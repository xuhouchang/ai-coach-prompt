# Coze adapter

Upload or package the complete skill folder with `SKILL.md` at the root. Before
practice, use Coze's current session capability in this order: safe HTML/card
renderer → Mermaid → `course_map.md` Markdown diagram. Use native choice/card
controls when the active Coze surface exposes them; otherwise use the
labelled-letter fallback. Do not require a desktop-client widget, local path,
or shell command for the learner flow.

If a Coze-specific manifest becomes required, add it in this adapter only rather than changing the portable core.
