# AI Coach Prompt

> 面向新手的 AI Prompt 实战教练：不是先学完再开始，而是在真实任务里，一边做，一边学。  
> **Learn in Practice｜在实践中学习**

AI Coach Prompt 是一款为 WorkBuddy / CodeBuddy 用户设计的交互式教学 Skill。它把“学会写 Prompt”从抽象的概念课，变成一段可跟着完成的实战练习：用户带着自己的工作目标进入，Skill 用短问题、分步引导、即时反馈和阶段测试，陪用户把模糊的想法逐渐写成可执行、可复用的 Prompt。

## 为什么需要它

很多刚开始使用 AI 的人并不是没有需求，而是不知道怎样把需求说清楚：

- 想让 AI 帮忙，却不知道从哪里开始描述；
- 看过很多“万能提示词”，放到自己的工作里依然不好用；
- 知道 Prompt 很重要，但缺少可以立刻动手的练习路径；
- 希望边完成真实工作，边建立自己的 AI 协作能力。

AI Coach Prompt 不把用户当成需要背知识点的学生，而是把每一个真实任务当成练习场。它不替用户完成业务任务；它教用户把任务讲清楚、拆清楚、验收清楚，让 AI 真正成为可协作的工作伙伴。

## 它能帮助用户做什么

通过一套循序渐进的学习流程，用户可以：

- 识别一个好 Prompt 必须包含的关键信息；
- 把模糊需求转化为清晰的任务说明；
- 学习六类常见 AI 任务的 Prompt 写法；
- 按节点一步步搭建自己的 Prompt，而不是面对空白输入框发愁；
- 查看合并后的完整 Prompt，并持续复习、优化与迁移；
- 用五维评分理解 Prompt 的优点与待改进之处。

## 核心理念：Learn in Practice

**先做，再学；边做，边会。**

传统课程往往先讲一大段理论，再让用户回到工作里自己尝试。AI Coach Prompt 反过来设计学习过程：

1. 从用户的 AI 使用经验、行业与目标出发，匹配合适的练习路线；
2. 用一个真实或贴近真实的工作场景，带用户进入任务；
3. 每次只完成一个小节点，例如目标、背景、输入、输出格式或验收标准；
4. 在完成中获得解释、示例与即时反馈；
5. 自动汇总为完整 Prompt，让学习成果能马上带回工作中使用。

这样，学习不是脱离工作的额外负担，而是完成工作的过程本身。

## 学习体验

```text
个性化问卷
  → Prompt 基础认知
  → 选择任务类型与练习场景
  → 分节点搭建 Prompt
  → 合并生成完整 Prompt
  → 五维测试与复盘
  → 继续学习下一类任务
```

整个过程采用“小步、单页、可继续”的交互方式：一次只解决一个问题，听不懂可以换例子，暂时跳过也能继续，并会保存学习进度。新手不需要一开始就写一大段 Prompt。

## 适合谁

- 刚开始在工作中使用 AI 的 WorkBuddy / CodeBuddy 用户；
- 想把零散 AI 尝试变成稳定工作方法的人；
- 希望用自己的任务练习 Prompt，而不只看模板的人；
- 需要为团队成员提供轻量、可操作 AI 培训的组织。

## 教学范围与边界

这是一个 **Prompt 写作与评测教练**，专注帮助用户学习如何表达任务、补全上下文和定义结果。

- 它不会代替用户执行 Prompt 中的业务任务；
- 用户提交的 Prompt 仅作为学习与评测材料，不会被执行；
- 课程会引导用户在安全、合规的边界内使用 AI。

## 开始使用

安装插件后，在 WorkBuddy / CodeBuddy 中手动调用：

```text
/ai-coach-prompt
```

无需准备长 Prompt。回答几个简短问题后，就可以开始第一段“边做边学”的练习。
